# github建站方法
###### 1、注册一个github账号，可以点击下方链接进入github
[点击进入github](https://www.github.com)
###### 2、点击右上方“+”
![image](https://user-images.githubusercontent.com/122024250/210945908-f7a577d4-18ae-4e32-8227-c602d42b4796.png)
###### 3、选择第一个“新建储存库”
![image](https://user-images.githubusercontent.com/122024250/210946105-7d53da31-b0f1-46f1-a2a9-feaaca9521d6.png)
###### 4、储存库名称输入[你的昵称].github.io，比如如果昵称叫abc，就输入abc.github.io
![image](https://user-images.githubusercontent.com/122024250/210946382-048b99f9-f6b5-42b9-bd28-edaceb2fa273.png)
###### 5、选择“公共”（默认就是）
![image](https://user-images.githubusercontent.com/122024250/210946492-40dd3f96-3a0d-4b07-9e55-19736583fac2.png)
###### 6、勾选Add a README file（默认没有勾）
![image](https://user-images.githubusercontent.com/122024250/210946849-41f049ae-9d60-4603-a81d-52cce363f774.png)
###### 7、点击新建
![image](https://user-images.githubusercontent.com/122024250/210946899-69f1cac5-3423-4b22-8598-d7227599371c.png)
###### 8、打开你的储存库的设置界面，在左侧列表选择“pages”
![image](https://user-images.githubusercontent.com/122024250/210947307-1767ea6d-a738-45d9-a63d-70a7bafb9e26.png)
![image](https://user-images.githubusercontent.com/122024250/210947458-975218d8-7bd0-442a-a7b7-accf83d6401d.png)
###### 9、将各种配置调整为如图所示
![image](https://user-images.githubusercontent.com/122024250/210947612-33c578c5-519b-4039-b342-f255c443c37e.png)
###### 10、回到Code界面，打开README.md并编辑
![image](https://user-images.githubusercontent.com/122024250/210947896-cb172726-f6a8-4bd3-be62-3d5295162328.png)
###### 11、用markdown语言进行编辑即可
###### 12、保存
![image](https://user-images.githubusercontent.com/122024250/210948033-17d110fb-bc86-4ee7-9a3e-b8b46f48d262.png)
###### 13、等待几分钟，再进入[你的昵称].github.io即可看到！
# 请注意：必须是[你的昵称].github.io，如果不创建以你的昵称命名的网站，将不会起作用！
